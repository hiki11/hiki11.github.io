C4D基础笔记

发表于2026-01-02|更新于2026-01-02|影视

一、 核心视图导航（鼠标+键盘组合）

这是在 3D 空间里“走路”的方法，建议养成左手按数字键、右手动鼠标的习惯：

  • Alt + 鼠标左键：旋转视图（Orbit）

  • Alt + 鼠标中键（滚轮按下）：平移视图（Pan）

  • Alt + 鼠标右键：缩放视图（Zoom/Dolly）

  • 鼠标中键（滚轮点击）：在“单视图”与“四视图”之间快速切换

  • H 键：最大化显示场景中所有物体

  • S 键：最大化显示当前选中的物体（非常有用，当模型找不着时必按）

二、 基础工具与变换

在进行建模或摆放物体时，这些快捷键是肌肉记忆：

功能 快捷键 备注
移动 E 调出位移坐标轴
旋转 R 调出旋转圆环
缩放 T 调出缩放手柄
全选 Ctrl + A 选中所有物体或元素
取消全选 Ctrl + Shift + A
转为可编辑 C 将参数化几何体“塌陷”为多边形（必记）
切换全屏 Ctrl + Tab 最大化当前视窗

三、 建模模式切换

C4D 的灵活性在于你可以在不同层级编辑物体，这些模式通常在左侧边栏：

  • Enter (回车)：在“对象模式”与“点/边/面模式”之间快速切换。
  • 点模式：数字键 1（注意：旧版本或部分配置中可能是左侧图标点击）。
  • 边模式：数字键 2。
  • 面模式：数字键 3。
  • 坐标系切换 (W)：在“世界坐标”和“物体坐标”之间切换。

四、 常用建模快捷键（组合键）

C4D 的建模命令大多是“两步式”快捷键（先按第一个字母弹出菜单，再按第二个字母执行）：

  • 常用菜单:

  • U ~ L：循环选择（Loop Selection），用于快速选中一圈线或面。

  • M ~ S：倒角（Bevel），给边缘增加圆角或细节。

  • M ~ T：挤压（Extrude），最常用的建模命令。

  • M ~ W：内部挤压（Inner Extrude）。

  • K ~ K：切刀工具（Knife），在模型上加线。

  • 打组与层级:

  • Alt + G：将选中的物体打组（放入空白对象下）。

  • Shift + G：取消打组。

  • Shift + 点击对象：将新对象创建为当前选中的子级。

  • Alt + 点击生成器：将当前对象直接放入生成器（如细分曲面）的子级。

五、 渲染与系统

  • Ctrl + R：在当前视口预览渲染。
  • Shift + R：渲染至图片查看器（最终输出前的检查）。
  • Ctrl + B：打开渲染设置。
  • Ctrl + D：工程设置（修改帧率、时长等）。
  • Ctrl + E：软件偏好设置。

💡 进阶小贴士

  1. Shift + C：C4D 的万能搜索框。
  2. 右键：如果你忘记了快捷键，在视口或对象栏点右键，几乎所有常用命令都在弹出的上下文菜单里。
  3. 属性面板：C4D 几乎所有的修改都在右下角的“属性”窗口完成，养成修改参数后点击“应用”或直接观察变化的习惯。
